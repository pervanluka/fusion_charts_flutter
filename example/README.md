# Fusion Charts Flutter Example

Interactive gallery showcasing all features of the `fusion_charts_flutter` package.

## Running the Example

```bash
cd example
flutter pub get
flutter run
```

## Features Demonstrated

- **5 Chart Types**: Line, Bar, Stacked Bar, Pie, Donut
- **Themes**: Light and Dark mode
- **Interactions**: Tooltips, Crosshair, Zoom & Pan
- **Customization**: Gradients, Shadows, Markers, Track Bars
- **Performance**: LTTB downsampling for large datasets
